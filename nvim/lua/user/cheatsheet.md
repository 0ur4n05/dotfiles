## Plugins
vim-fugitive : allows you to do all o the vim commands only from the vim command line
comment.nvim : commenting made easy
	[how to use]
		gcc : one line comment
		gbc : multiple line comments
		<number><command> : comment a lot of lines
		// visual mode
		gc/gb : comments a lot of lines selected

chadtree : a files manager in neovim
	[how to use]
		cc : close/open chadtree
42header : 42 header nvim
	[how to use]
		std: insert header
date : 
	[how to use]
		da : shows you date
man : shows the man of a function
	[how to use]
		:Man <function>
